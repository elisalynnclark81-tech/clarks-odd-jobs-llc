GitHub Pages Hosting
Host this dashboard on GitHub Pages so you can access it from any phone, tablet, or computer — no app store needed. Add it to your phone home screen for a full app experience.
1
Go to github.com → sign in or create a free account
2
Click New Repository → name it clarks-odd-jobs → set Public → Create
3
Click Add file → Upload files → upload this index.html file
4
Go to Settings → Pages → Source: Deploy from branch → main → / (root) → Save
5
Your dashboard will be live at yourusername.github.io/clarks-odd-jobs
6
On your iPhone/iPad: open Safari → navigate to the URL → tap Share → Add to Home Screen
Note: Data is saved locally on each device. Connect Google Sheets sync to share data across devices.
