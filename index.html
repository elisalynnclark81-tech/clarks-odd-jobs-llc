Google Sheets Sync
Connect Clark's Odd Jobs to Google Sheets for automatic data sync. All clients, jobs, and invoices will write to your sheet in real time.
Setup Instructions
1. Open Google Sheets → create a new sheet called ClarksOddJobs
2. Go to Extensions → Apps Script
3. Paste the Apps Script code (download below)
4. Click Deploy → New Deployment → Web App
5. Set "Execute as: Me" and "Who has access: Anyone"
6. Copy the Web App URL and paste it above
