# Clark's Odd Jobs — GitHub Pages Setup

## What You Get
A live dashboard at: **yourusername.github.io/clarks-odd-jobs**
Works on phone, tablet, and desktop. Add to iPhone/iPad home screen for a full app feel.

---

## Step 1: Create a GitHub Account
Go to **github.com** → Sign Up (free). Use any email.

---

## Step 2: Create the Repository
1. Click the **+** icon (top right) → **New Repository**
2. Repository name: `clarks-odd-jobs`
3. Set to **Public**
4. Click **Create Repository**

---

## Step 3: Upload Your Files
1. Click **Add file → Upload files**
2. Upload all 4 files:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `GITHUB_SETUP.md` (optional)
3. Click **Commit changes**

---

## Step 4: Enable GitHub Pages
1. Go to **Settings** tab (in your repo)
2. Scroll to **Pages** in the left menu
3. Under **Source**: select `Deploy from a branch`
4. Branch: **main** — Folder: **/ (root)**
5. Click **Save**

Wait 1–2 minutes, then your site is live at:
`https://yourusername.github.io/clarks-odd-jobs`

---

## Step 5: Add to iPhone Home Screen
1. Open **Safari** on your iPhone/iPad
2. Navigate to your GitHub Pages URL
3. Tap the **Share** button (box with arrow up)
4. Scroll down → tap **Add to Home Screen**
5. Name it **Clark's Odd Jobs** → tap **Add**

Now it works like a native app with your orange icon.

---

## Step 6: Set Up Google Sheets Sync
See the **Google Sheets Sync** tab inside the dashboard for full instructions.
This lets your data sync across your phone AND computer.

---

## Updating the Dashboard Later
When you get an updated `index.html`:
1. Go to your GitHub repo
2. Click **index.html** → click the **pencil** (edit) icon
3. Or click **Add file → Upload files** and re-upload
