# Clark's Odd Jobs — Business Dashboard

Interactive live app for managing a lawn care and odd jobs business.

**Live:** https://elisalynnclark81-tech.github.io/claudecowork/

## Features

- **Dashboard** — stats, today's jobs, upcoming schedule, monthly progress
- **Today's Route** — ordered stops with Apple Maps directions & one-tap calling
- **Schedule** — interactive monthly calendar with color-coded tasks
- **Clients** — full CRUD with service preferences and frequency tracking
- **Jobs** — create, track, and complete jobs with task breakdown
- **Invoices** — generate printable invoices, mark paid/unpaid, Google Sheets sync
- **Settings** — business info, tax rate, data export/import

## Tech

Single-file vanilla JS + CSS, no dependencies. localStorage for persistence, installable PWA.
